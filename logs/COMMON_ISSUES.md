## 🤖 Ошибки Агентов

### 1. Превышен лимит токенов

**Симптомы:**
- Ошибка: `TokenLimitExceeded`
- Агент: Обычно Manager или Engineer
- Когда: Длинные разговоры с клиентом

**Причина:**
Слишком много сообщений в контексте

**Решение:**
1. Сжать контекст (оставить последние 10 сообщений)
2. Создать краткое резюме требований
3. Если не помогло - разбить на подзапросы

**Код для предотвращения:**
```python
if conversation_length > 15:
    context = compress_context(messages)
```

## 🗄️ Ошибки Базы Данных

### 1. Timeout при подключении к Supabase

**Симптомы:**
- `TimeoutError: Connection timeout`
- Происходит случайно

**Причина:**
- Нестабильное интернет-соединение
- Supabase перегружен

**Решение:**
1. Retry с экспоненциальной задержкой (1s, 2s, 4s)
2. Если 3 попытки не помогли - уведомить админа

**Код:**
```python
@retry(max_attempts=3, backoff=exponential)
def connect_to_supabase():
    return supabase_client.connect()
```

**Статус:** ✅ Решено
**Дата:** 2024-01-15

## 🔌 Ошибки API

### 1. Claude API: Rate limit exceeded

**Симптомы:**
- `RateLimitError: Too many requests`
- Время: обычно в пиковые часы

**Причина:**
Превышен лимит запросов в минуту

**Решение:**
1. Добавить очередь запросов
2. Задержка между запросами 200ms
3. Кешировать повторяющиеся запросы

**Код:**
```python
rate_limiter = RateLimiter(max_per_minute=30)
result = rate_limiter.execute(claude_api_call)
```

### 2. Gemini API: Invalid image format

**Симптомы:**
- `InvalidImageError: Unsupported format`
- Vision не может обработать скриншот

**Причина:**
- Скриншот в формате WEBP или HEIC
- Размер > 10MB

**Решение:**
1. Конвертировать в JPEG/PNG
2. Сжать если > 5MB

**Код:**
```python
from PIL import Image

def prepare_image(image_path):
    img = Image.open(image_path)
    if img.format not in ['JPEG', 'PNG']:
        img = img.convert('RGB')
    img.save('temp.jpg', quality=85)
```

**Причина:**

1. Добавить cron задачу: проверять заказы каждый час
2. Если таймаут истек → перевести в `cancelled`

**Код:**
```python
def check_timeouts():
    stuck_orders = get_orders_with_timeout()
```
