# Пояснение папок и файлов проекта

## Файлы в корне

| Файл | Что это |
|------|---------|
| `main.py` | Точка входа — запускает бота |
| `.env` | Секретные настройки (токены, ключи). **Не трогать, не показывать никому** |
| `.env.example` | Шаблон для `.env` — показывает какие переменные нужны |
| `requirements.txt` | Список Python-библиотек, которые нужны для работы |
| `CLAUDE.md` | Инструкции для Claude Code — как работать с этим проектом |
| `КОМАНДЫ.md` | Этот файл — список нужных команд |
| `README.md` | Общее описание проекта |
| `.gitignore` | Список файлов, которые не попадают в Git (например `.env`) |

---

## 📁 config/ — Настройки

| Файл | Что это |
|------|---------|
| `settings.py` | Читает `.env` и отдаёт настройки всему проекту |
| `prompts.py` | Системные промпты для будущих AI-агентов (пока не используются) |
| `agents_config.yaml` | Конфигурация будущих 6 агентов (пока не используется) |

## 📁 src/ — Весь код проекта

### src/integrations/telegram/
| Файл | Что это |
|------|---------|
| `bot.py` | **Главный файл бота** — весь диалог с клиентом и админом |

### src/core/
| Файл/папка | Что это |
|------------|---------|
| `database.py` | Подключение к Supabase, все запросы к базе данных |
| `agents/` | Пусто — здесь будут AI-агенты (следующие этапы) |
| `orchestrator/` | Пусто — здесь будет управление агентами |
| `services/` | Пусто — здесь будут вспомогательные сервисы |

### src/integrations/ (остальные папки)
| Папка | Что это |
|-------|---------|
| `llm/` | Пусто — здесь будет интеграция с языковыми моделями |
| `database/` | Пусто — резерв для будущего |
| `storage/` | Пусто — здесь будет хранение файлов/изображений |
