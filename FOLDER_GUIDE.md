# Repository Guide

- `config/` contains settings, agent configuration, and system prompts.
- `src/` contains the Telegram integration and Supabase data layer.
- `scripts/` contains setup and reporting utilities.
- `deployment/` contains the Docker deployment definition.
- `logs/` contains log documentation and an error-record template.
- `docs/`, `audience/`, and `competitors/` contain product and technical documentation.
