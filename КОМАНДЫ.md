# Команды проекта

## Как открыть терминал (консоль)
В VS Code: нажми **Ctrl+`** (тильда — клавиша слева от цифры 1)
Или через меню: **Terminal → New Terminal**

## Запуск бота
```
python main.py
```

## Остановка бота
```
Ctrl+C
```

## Установка зависимостей
```
python -m pip install -r requirements.txt
```

## Настройка базы данных
Выводит SQL — нужно вставить в Supabase SQL Editor и выполнить:
```
python scripts/setup_database.py
```

## Docker
```
docker build -t ai-viz-bot .
docker run --env-file .env ai-viz-bot
```
